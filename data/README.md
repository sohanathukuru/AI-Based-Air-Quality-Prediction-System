# Dataset

The datasets used in this project are not included in this repository because they exceed GitHub's file size limitations.

## Raw Datasets

The following raw datasets were used during development:

- station_hour.csv
- Delhi_Hyderabad_Station_Hour.csv
- delhi_ncr_aqi_dataset.csv
- stations.csv

## Processed Datasets

The following datasets were generated during preprocessing and feature engineering:

- preprocessed_aqi_dataset.csv
- feature_engineered_dataset.csv

## Dataset Source

The primary dataset used in this project was obtained from Kaggle.

Original data source:
Central Pollution Control Board (CPCB), Government of India.

The processed datasets included in this project were generated using the preprocessing and feature engineering pipeline implemented in the source code.
